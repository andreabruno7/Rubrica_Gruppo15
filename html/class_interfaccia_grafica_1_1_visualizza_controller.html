◆ inizializzaVisualizza()

void InterfacciaGrafica.VisualizzaController.inizializzaVisualizza ( Contatto contatto)

Inizializza gli elementi grafici della finestra di visualizzazione.

Il metodo inizializza le Label relative alle informazioni del contatto con i campi del contatto passato come parametro.

Precondition
Il controller deve essere associato al file fxml opportuno.
Postcondition
Vengono visualizzate nella finestra tutte le informazioni del contatto.
Parameters
[in]contattoIl contatto da visualizzare.
