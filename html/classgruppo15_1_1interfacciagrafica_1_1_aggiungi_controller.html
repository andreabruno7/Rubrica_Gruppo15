◆ inizializzaAggiungi()

void gruppo15.interfacciagrafica.AggiungiController.inizializzaAggiungi ( )

Inizializza gli elementi grafici della finestra per aggiungere i contatti.

Vengono inizializati i TextField come vuoti. Non è possibile aggiungere un contatto che non abbia almeno il nome o il cognome.

Precondition
Il controller deve essere associato al file fxml opportuno.

◆ isAggiunto()

boolean gruppo15.interfacciagrafica.AggiungiController.isAggiunto ( )

Controlla se il contatto è stato aggiunto.

Il metodo ritorna true se il contatto mantenuto dal controller deve essere aggiunto alla rubrica, altrimenti ritorna false.

Returns
Viene restituito true se il contatto mantenuto dal controller deve essere aggiunto , altrimenti ritorna false.
