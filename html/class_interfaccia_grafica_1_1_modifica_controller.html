◆ inizializzaModifica()

void InterfacciaGrafica.ModificaController.inizializzaModifica ( Contatto contatto)

Inizializza gli elementi grafici della finestra di modifica.

Viene inizializzato il contatto da modificare e vengono mostrate le sue informazioni nei TextField. Il contatto che viene passato non è stato ancora modificato.

Precondition
Il controller deve essere associato al file fxml opportuno.
Postcondition
Vengono visualizzate nella finestra tutte le informazioni del contatto.
Queste informazioni sono modificabili.
Parameters
[in]contattoil contatto da modificare.

Controlla se il contatto è stato modificato.

Il metodo ritorna true se il contatto mantenuto dal controller deve essere modificato , altrimenti ritorna false.

Returns
Viene restituito true se il contatto mantenuto dal controller deve essere modificato , altrimenti ritorna false.
